Note 5.7. Rule: Right-Endpoint Approximation.

Construct a rectangle on each subinterval \([x_{i-1} ,x_i],\) only this time the height of the rectangle is determined by the function value \(f(x_i)\) at the right endpoint of the subinterval. Then, the area of each rectangle is \(f(x_i)\Delta x \) and the approximation for \(A\) is given by

\begin{align*} A\approx R_n\amp =f(x_1)\Delta x +f(x_2)\Delta x + \cdots +f(x_n)\Delta x \\ \amp=\displaystyle{ \sum_{i=1}^{n} f(x_i)\Delta x }. \end{align*}
