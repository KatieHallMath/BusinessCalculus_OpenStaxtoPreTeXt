Recall that a function has exactly one output for each input. Therefore, to define an inverse function, we need to map each input to exactly one output. For example, letâ€™s try to find the inverse function for \(f(x)=x^2.\) Solving the equation \(y=x^2\) for \(x,\) we arrive at the equation \(x=\pm \sqrt{y}.\) This equation does not describe \(x\) as a function of \(y\) because there are two solutions to this equation for every \(y\gt 0.\) The problem with trying to find an inverse function for \(f(x)=x^2\) is that two inputs are sent to the same output for each output \(y\gt 0.\) The function \(f(x)=x^3+4\) discussed earlier did not have this problem. For that function, each input was sent to a different output. A function that sends each input to a different output is called a one-to-one function.
