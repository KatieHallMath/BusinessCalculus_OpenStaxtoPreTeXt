Solution 4.168.1.

  1. Using Theorem 4.167, we can integrate each of the four terms in the integrand separately. We obtain

    \begin{equation*} \int (5x^3-7x^2+3x+4)dx = \int 5x^3dx- \int 7x^2dx + \int 3x dx + \int 4dx . \end{equation*}
    From the second part of Theorem 4.167, each coefficient can be written in front of the integral sign, which gives
    \begin{equation*} \int 5x^3dx- \int 7x^2dx + \int 3x dx + \int 4dx =5 \int x^3dx-7 \int x^2dx +3 \int x dx +4 \int 1dx. \end{equation*}
    Using the power rule for integrals, we conclude that
    \begin{equation*} \int (5x^3-7x^2+3x+4)dx =\frac{5}{4}x^4-\frac{7}{3}x^3+\frac{3}{2}x^2+4x+C. \end{equation*}

  2. Rewrite the integrand as

    \begin{equation*} \frac{x^2+4\sqrt[3]{x} }{x}=\frac{x^2}{x}+\frac{4\sqrt[3]{x}}{x}. \end{equation*}
    Then, to evaluate the integral, integrate each of these terms separately. Using the power rule, we have
    \begin{align*} \int (x+\frac{4}{x^{2/3}}) dx\amp = \int x dx+4 \int x^{-2/3} dx\\ \amp =\frac{1}{2}x^2+4\frac{1}{(\frac{-2}{3})+1}x^{(-2/3)+1}+C\\ \amp =\frac{1}{2}x^2+12x^{1/3}+C. \end{align*}

  3. Rewrite the integrand as

    \begin{equation*} (x+1)(x+2)=x^2+3x+2. \end{equation*}
    Therefore,
    \begin{equation*} \int (x+1)(x+2) dx = \int x^2+3x+2 dx =\frac{1}{3}x^3+\frac{3}{2}x^2+2x+C. \end{equation*}

  4. Using Theorem 4.167, we can integrate each of the two terms in the integrand separately. We obtain

    \begin{equation*} \int (2^x+7e^x)dx = \int 2^xdx+\int 7e^xdx . \end{equation*}
    From the second part of Theorem 4.167, each coefficient can be written in front of the integral sign, which gives
    \begin{equation*} \int 2^xdx+7\int e^xdx. \end{equation*}
    Using Table 4.164, we conclude that
    \begin{equation*} \int (2^x+7e^x)dx = \frac{2^x}{\ln 2}+7e^x+C. \end{equation*}
