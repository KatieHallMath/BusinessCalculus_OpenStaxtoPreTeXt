Solution 1.63.1.

  1. You cannot divide by zero, so the domain is the set of values \(x\) such that \(x^2 − 1 \neq 0 .\) Therefore, the domain is \({x|x\neq \pm 1 }.\)

  2. You need to determine the values of \(x\) for which the denominator is zero. Since \(3 x^2 + 4 \geq 4 \) for all real numbers \(x,\) the denominator is never zero. Therefore, the domain is \((-\infty,\infty).\)

  3. Since the square root of a negative number is not a real number, the domain is the set of values \(x\) for which \(4 − 3 x\geq 0 .\) Therefore, the domain is \({x|x\leq 4 / 3 }.\)

  4. The cube root is defined for all real numbers, so the domain is the interval \((-\infty,\infty)\)
