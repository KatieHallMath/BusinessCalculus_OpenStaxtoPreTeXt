Solution 4.12.1.

Since we are looking for the linear approximation at \(x=9,\) using (4.2.1) we know the linear approximation is given by

\begin{equation*} L(x)=f(9)+f'(9)(x-9). \end{equation*}

We need to find \(f(9)\) and \(f'(9).\)

\begin{align*} f(x)=\sqrt{x}\amp \implies \amp \amp f(9)=\sqrt{9}=3\\ f'(x)=\frac{1}{2\sqrt{x}}\amp\implies \amp \amp f'(9)=\frac{1}{2\sqrt{9}}=\frac{1}{6} \end{align*}

Therefore, the linear approximation is given by FigureÂ 4.13.

\begin{equation*} L(x)=3+\frac{1}{6}(x-9) \end{equation*}

Using the linear approximation, we can estimate \(\sqrt{9.1}\) by writing

\begin{equation*} \sqrt{9.1}=f(9.1)\approx L(9.1)=3+\frac{1}{6}(9.1-9)\approx 3.0167. \end{equation*}
