Theorem 4.59. First Derivative Test.

Suppose that \(f\) is a continuous function over an interval \(I\) containing a critical point \(c.\) If \(f\) is differentiable over \(I,\) except possibly at point \(c,\) then \(f(c)\) satisfies one of the following descriptions:

  1. If \(f'\) changes sign from positive when \(x\lt c\) to negative when \(x\gt c,\) then \(f(c)\) is a local maximum of \(f.\)

  2. If \(f'\) changes sign from negative when \(x\lt c\) to positive when \(x\gt c,\) then \(f(c)\) is a local minimum of \(f.\)

  3. If \(f\) has the same sign for \(x\lt c\) and \(x\gt c,\) then \(f(c)\) is neither a local maximum nor a local minimum of \(f.\)
