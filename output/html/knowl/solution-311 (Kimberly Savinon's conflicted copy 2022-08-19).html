Solution 5.57.1.

The formula states the mean value of \(f(x)\) is given by

\begin{equation*} \frac{1}{4-0}\displaystyle{ \int_{0}^{4} (8-2x)dx}. \end{equation*}

We can see in Figure 5.58 that the function represents a straight line and forms a right triangle bounded by the \(x\)- and \(y\)-axes. The area of the triangle is \(A=\frac{1}{2}(\text{ base })}{(\text{ height }).\) We have

\begin{equation*} A=\frac{1}{2}(4)}{(8)=16. \end{equation*}

The average value is found by multiplying the area by \(1/(4-0).\) Thus, the average value of the function is

\begin{equation*} \frac{1}{4}(16)=4. \end{equation*}

Set the average value equal to \(f(c)\) and solve for \(c\text{.}\)

\begin{align*} 8-2c\amp =\amp 4\\ c\amp =\amp 2 \end{align*}

At \(c=2,f(2)=4.\)
