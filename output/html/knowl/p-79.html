A composite function \(g\circ f\) can be viewed in two steps. First, the function \(f\) maps each input \(x\) in the domain of \(f\) to its output \(f(x)\) in the range of \(f.\) Second, since the range of \(f\) is a subset of the domain of \(g,\) the output \(f(x)\) is an element in the domain of \(g,\) and therefore it is mapped to an output \(g(f(x))\) in the range of \(g.\) In FigureÂ 1.32, we see a visual image of a composite function.
