Recall the properties of exponents: If \(x\) is a positive integer, then we define \(b^x=b\cdot b\cdots b\) (with \(x\) factors of \(b).\) If \(x\) is a negative integer, then \(x=.y\) for some positive integer \(y,\) and we define \(b^x=b^{-y}=1/b^y.\) Also, \(b^0\) is defined to be \(1.\) If \(x\) is a rational number, then \(x=p/q,\) where \(p\) and \(q\) are integers and \(b^x=b^{p/q}=\sqrt[q]{b^p}.\) For example, \(9^{3/2}=\sqrt{9^3}=27.\) However, how is \(b^x\) defined if \(x\) is an irrational number? For example, what do we mean by \(2^\sqrt{2}\) This is too complex a question for us to answer fully right now; however, we can make an approximation. We claim that if we choose rational numbers \(x\) getting closer and closer to \(\sqrt{2},\) the values of \(2^x\) get closer and closer to some number \(L.\) We define that number \(L\) to be \(2^\sqrt{2}.\)
