Item d.
Using Theorem 4.167, we can integrate each of the two terms in the integrand separately. We obtain
\begin{equation*} \int (2^x+7e^x)dx = \int 2^xdx+\int 7e^xdx . \end{equation*}
From the second part of Theorem 4.167, each coefficient can be written in front of the integral sign, which gives
\begin{equation*} \int 2^xdx+7\int e^xdx. \end{equation*}
Using Table 4.164, we conclude that
\begin{equation*} \int (2^x+7e^x)dx = \frac{2^x}{\ln 2}+7e^x+C. \end{equation*}
