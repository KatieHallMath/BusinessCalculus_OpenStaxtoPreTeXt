Some polynomial functions are power functions. A power function is any function of the form \(f(x)=ax^b ,\) where \(a\) and \(b\) are any real numbers. The exponent in a power function can be any real number, but here we consider the case when the exponent is a positive integer. (We consider other cases later.) If the exponent is a positive integer, then \(f(x)=ax^n\) is a polynomial. If \(n\) is even, then \(f(x)=ax^n\) is an even function because \(f(-x)=a(-x)^n=ax^n\) if \(n\) is even. If \(n\) is odd, then \(f(x)=ax^n\) is an odd function because \(f(-x)=a(-x)^n=-ax^n\) if \(n\) is odd (FigureÂ 1.55).
