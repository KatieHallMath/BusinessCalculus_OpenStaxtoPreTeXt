Solution 5.10.1.

First, divide the interval \([0,2]\) into \(n\) equal subintervals. Using \(n=4,\Delta x =\frac{(2-0)}{4}=0.5.\) This is the width of each rectangle. The intervals \([0,0.5],[0.5,1],[1,1.5],[1.5,2]\) are shown in Figure 5.11. Using a left-endpoint approximation, the heights are \(f(0)=0,f(0.5)=0.25,f(1)=1,f(1.5)=2.25.\) Then,

\begin{align*} L_4\amp =f(x_0)\Delta x +f(x_1)\Delta x +f(x_2)\Delta x +f(x_3)\Delta x \\ \amp =0(0.5)+0.25(0.5)+1(0.5)+2.25(0.5)\\ \amp =1.75. \end{align*}

The right-endpoint approximation is shown in Figure 5.12. The intervals are the same, \(\Delta x =0.5,\) but now use the right endpoint to calculate the height of the rectangles. We have

\begin{align*} R_4\amp =f(x_1)\Delta x +f(x_2)\Delta x +f(x_3)\Delta x +f(x_4)\Delta x \\ \amp =0.25(0.5)+1(0.5)+2.25(0.5)+4(0.5)\\ \amp =3.75. \end{align*}

The left-endpoint approximation is 1.75; the right-endpoint approximation is 3.75.
