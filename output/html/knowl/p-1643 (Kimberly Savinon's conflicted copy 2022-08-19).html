On each subinterval \([x_{i-1} ,x_i]\) (for \(i=1,2,3 , \dots , n),\) construct a rectangle with width \(\Delta x \) and height equal to \(f(x_{i-1} ),\) which is the function value at the left endpoint of the subinterval. Then the area of this rectangle is \(f(x_{i-1} )\Delta x .\) Adding the areas of all these rectangles, we get an approximate value for \(A\) (FigureÂ 5.6). We use the notation L_n to denote that this is a left-endpoint approximation of \(A\) using \(n\) subintervals.
