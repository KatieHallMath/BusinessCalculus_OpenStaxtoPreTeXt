First we need to solve the differential equation. If \(\frac{dy}{dx}=2e^x,\) then

\begin{equation*} y= \int 2e^x dx =2e^x+C. \end{equation*}

Next we need to look for a solution \(y\) that satisfies the initial condition. The initial condition \(y(0)=5\) means we need a constant \(C\) such that \(2e^0+C=5.\) Therefore,

\begin{equation*} C=5-2e^0=3. \end{equation*}

The solution of the initial-value problem is \(y=2e^x+3.\)
