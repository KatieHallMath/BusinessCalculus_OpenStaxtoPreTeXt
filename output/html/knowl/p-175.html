The root functions \(f(x)= x^{1/n}\) have defining characteristics depending on whether \(n\) is odd or even. For all even integers \(n\geq 2 ,\) the domain of \(f(x)= x^{1/n}\) is the interval \([ 0 ,\infty).\) For all odd integers \(n\geq 1 ,\) the domain of \(f(x)= x^{1/n}\) is the set of all real numbers. Since \(x^{1/n}=-(-x)^{1/n}\) for odd integers \(n,f(x)= x^{1/n}\) is an odd function if \(n\) is odd. See the graphs of root functions for different values of \(n\) in FigureÂ 1.62.
