A vertical scaling of a graph occurs if we multiply all outputs \(y\) of a function by the same positive constant. For \(c\gt 0 ,\) the graph of the function \(cf(x)\) is the graph of \(f(x)\) scaled vertically by a factor of \(c.\) If \(c\gt 1 ,\) the values of the outputs for the function \(cf(x)\) are larger than the values of the outputs for the function \(f(x);\) therefore, the graph has been stretched vertically. If \(0 \lt c\lt 1 ,\) then the outputs of the function \(cf(x)\) are smaller, so the graph has been compressed. For example, the graph of the function \(f(x)= 3 x^2 \) is the graph of \(y=x^2 \) stretched vertically by a factor of 3, whereas the graph of \(f(x)=x^2 / 3 \) is the graph of \(y=x^2 \) compressed vertically by a factor of \(3 \) (FigureÂ 1.74).
