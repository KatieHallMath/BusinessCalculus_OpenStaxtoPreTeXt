We know that if a continuous function has a local extrema, it must occur at a critical point. However, a function need not have a local extrema at a critical point. Here we examine how the second derivative test can be used to determine whether a function has a local extremum at a critical point. Let \(f\) be a twice-differentiable function such that \(f'(a)=0\) and \(f''\) is continuous over an open interval \(I\) containing \(a.\) Suppose \(f''(a)\lt 0.\) Since \(f''\) is continuous over \(I,\) \(f''(x)\lt 0\) for all \(x∈I\) (Figure 4.80). Then, by Corollary \(3,\) \(f'\) is a decreasing function over \(I.\) Since \(f'(a)=0,\) we conclude that for all \(x∈I,f'(x)\gt 0\) if \(x\lt a\) and \(f'(x)\lt 0\) if \(x\gt a.\) Therefore, by the first derivative test, \(f\) has a local maximum at \(x=a.\) On the other hand, suppose there exists a point \(b\) such that \(f'(b)=0\) but \(f''(b)\gt 0.\) Since \(f''\) is continuous over an open interval \(I\) containing \(b,\) then \(f''(x)\gt 0\) for all \(x∈I\) (Figure 4.80). Then, by Corollary \(3,f'\) is an increasing function over \(I.\) Since \(f'(b)=0,\) we conclude that for all \(x∈I,\) \(f'(x)\lt 0\) if \(x\lt b\) and \(f'(x)\gt 0\) if \(x\gt b.\) Therefore, by the first derivative test, \(f\) has a local minimum at \(x=b.\)
