Solution 4.94.1.
  1. Using the algebraic limit laws, we have \(\displaystyle{\lim_{x\to\infty}\left(5-\frac{2}{x^2}\right)=\lim_{x\to\infty}5-2\left(\lim_{x\to\infty}\frac{1}{x}\right).\left(\lim_{x\to\infty}\frac{1}{x}\right)=5-2\cdot0=5.}\) Similarly, \(\displaystyle{\lim_{x\to-\infty}f(x)=5.}\) Therefore, \(f(x)=5-\frac{2}{x^2}\) has a horizontal asymptote of \(y=5\) and \(f\) approaches this horizontal asymptote as \(x\to\pm\infty\) as shown in the following graph.

  2. Using the algebraic limit laws, we have \(\displaystyle{\lim_{x\to\infty}\left(2+\frac{3}{x}\right)=\lim_{x\to\infty}2+3\left(\lim_{x\to\infty}\frac{1}{x}\right)=2+2\cdot0=2.}\) Similarly, \(\displaystyle{\lim_{x\to-\infty}f(x)=2.}\) Therefore, \(f(x)=2+\frac{3}{x}\) has a horizontal asymptote of \(y=2\) and \(f\) approaches this horizontal asymptote as \(x\to\pm\infty\) as shown in the following graph.
