Suppose, however, that we have a function \(v(t)\) that gives us the speed of an object at any time \(t\text{,}\) and we want to find the object’s average speed. The function \(v(t)\) takes on an infinite number of values, so we can’t use the process just described. Fortunately, we can use a definite integral to find the average value of a function such as this.
