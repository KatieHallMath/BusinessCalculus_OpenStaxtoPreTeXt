Solution 1.115.1.

  1. Evaluating \(\sin^{-1} (-\sqrt{3}/2)\) is equivalent to finding the angle \(\theta \) such that \(\sin\,\theta =-\sqrt{3}/2\) and \(-\pi/2\leq \theta \leq \pi/2.\) The angle \(\theta =-\pi/3\) satisfies these two conditions. Therefore, \(\sin^{-1} (-\sqrt{3}/2)=-\pi/3.\)

  2. First we use the fact that \(\tan^{-1} (−1/\sqrt{3})=-\pi/6.\) Then \(\text{ tan }(\pi/6)=−1/\sqrt{3}.\) Therefore, \(\text{ tan }(\tan^{-1} (−1/\sqrt{3}))=−1/\sqrt{3}.\)

  3. To evaluate \(\cos^{-1}(\text{ cos }(5\pi/4)),\) first use the fact that \(\text{ cos }(5\pi/4)=-\sqrt{2}/2.\) Then we need to find the angle \(\theta \) such that \(\text{ cos }(\theta )=-\sqrt{2}/2\) and \(0\leq \theta \leq \pi.\) Since \(3\pi/4\) satisfies both these conditions, we have \(\text{ cos }(\cos^{-1}(5\pi/4))=\text{ cos }(\cos^{-1}(-\sqrt{2}/2))=3\pi/4.\)

  4. Since \(\text{ cos }(2\pi/3)=−1/2,\) we need to evaluate \(\sin^{-1} (−1/2).\) That is, we need to find the angle \(\theta \) such that \(\sin(\theta )=−1/2\) and \(-\pi/2\leq \theta \leq \pi/2.\) Since \(-\pi/6\) satisfies both these conditions, we can conclude that \(\sin^{-1} (\text{ cos }(2\pi/3))=\sin^{-1} (−1/2)=-\pi/6.\)
