Item 3.
We can find a formula for \((f\circ g)(x)\) in two ways. First, we could write
\begin{equation*} (f\circ g)(x)=f(g(x))=f\left(\frac{1}{x}\right)=\left(\frac{1}{x}\right)^2+1. \end{equation*}
Alternatively, we could write
\begin{equation*} (f\circ g)(x)=f(g(x))=(g(x))^2+1=\left(\frac{1}{x}\right)^2+1. \end{equation*}
The domain of \(f\circ g\) is the set of all real numbers \(x\) such that \(x\neq 0.\) To find the range of \(f,\) we need to find all values \(y\) for which there exists a real number \(x\neq0\) such that
\begin{equation*} \left(\frac{1}{x}\right)^2+1=y. \end{equation*}
Solving this equation for \(x,\) we see that we need \(x\) to satisfy
\begin{equation*} \left(\frac{1}{x}\right)^2=y-1, \end{equation*}
which simplifies to
\begin{equation*} \frac{1}{x}=\pm\sqrt{y-1}. \end{equation*}
Finally, we obtain
\begin{equation*} x=\pm\frac{1}{\sqrt{y-1}}. \end{equation*}
Since \(1/\sqrt{y-1}\) is a real number if and only if \(y\gt 1,\) the range of \(f\) is the set \(\{y|y\gt 1\}.\)
