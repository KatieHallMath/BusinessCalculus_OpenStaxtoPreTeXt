The easiest type of function to consider is a linear function. Linear functions have the form \(f(x)=ax+b,\) where \(a\) and \(b\) are constants. In FigureÂ 1.47, we see examples of linear functions when \(a\) is positive, negative, and zero. Note that if \(a\gt 0 ,\) the graph of the line rises as \(x\) increases. In other words, \(f(x)=ax+b\) is increasing on \((-\infty,\infty).\) If \(a\lt 0 ,\) the graph of the line falls as \(x\) increases. In this case, \(f(x)=ax+b\) is decreasing on \((-\infty,\infty).\) If \(a= 0 ,\) the line is horizontal.
