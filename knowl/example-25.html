Example 1.95. Maximizing Profit and Minimizing Cost.

  1. Find the revenue function \(R(x)\text{.}\)

  2. Find the profit function \(P(x)\text{.}\)

  3. How many dresses must be sold in order to maximize profits?

  4. What is the maximum profit?

  5. How many dresses must be sold in order to minimize costs?
