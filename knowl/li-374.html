Item 2.
Compute \(\displaystyle{\lim_{x\to a}f(x).}\) In some cases, we may need to do this by first computing \(\displaystyle{\lim_{x\to a^-}f(x)}\) and \(\displaystyle{\lim_{x\to a^+}f(x).}\) If \(\displaystyle{\lim_{x\to a}f(x)}\) does not exist (that is, it is not a real number), then the function is not continuous at \(a\) and the problem is solved. If \(\lim_{x\to a}f(x)\) exists, then continue to step 3.
