Item 4.
Using a graphing calculator or computer software that allows us graph functions, we can plot the function \(f(x),\) making sure the functional values of \(f(x)\) for \(x\)-values near \(a \) are in our window. We can use the trace feature to move along the graph of the function and watch the y-value readout as the \(x\)-values approach \(a \text{.}\) If the y-values approach \(L \) as our \(x\)-values approach \(a \) from both directions, then \(\lim_{x \to a} f(x)=L.\) We may need to zoom in on our graph and repeat this process several times.
